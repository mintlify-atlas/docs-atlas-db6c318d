## Overview

The Orders API provides endpoints for creating and managing customer orders. All order endpoints require authentication.

## GET /api/orders

Retrieve all orders for the authenticated user.

### Request

**Method:** `GET`

**Path:** `/api/orders`

**Authentication:** JWT token required

**Headers:**
- `Authorization: Bearer <access_token>`

### Response

<ResponseField name="orders" type="array">
  Array of order objects for the current user
  
  <ResponseField name="orders[].id" type="integer">
    Order's unique identifier
  </ResponseField>
  
  <ResponseField name="orders[].user_id" type="integer">
    User ID who placed the order
  </ResponseField>
  
  <ResponseField name="orders[].status" type="string">
    Order status (pending, paid, shipped, delivered, cancelled)
  </ResponseField>
  
  <ResponseField name="orders[].subtotal" type="number">
    Order subtotal before tax and discounts
  </ResponseField>
  
  <ResponseField name="orders[].tax" type="number">
    Tax amount
  </ResponseField>
  
  <ResponseField name="orders[].discount_amount" type="number">
    Discount amount applied
  </ResponseField>
  
  <ResponseField name="orders[].total" type="number">
    Total order amount
  </ResponseField>
  
  <ResponseField name="orders[].discount_code" type="string">
    Discount code used (if any)
  </ResponseField>
  
  <ResponseField name="orders[].created_at" type="datetime">
    Order creation timestamp
  </ResponseField>
  
  <ResponseField name="orders[].updated_at" type="datetime">
    Last update timestamp
  </ResponseField>
  
  <ResponseField name="orders[].items" type="array">
    Array of order items
    
    <ResponseField name="orders[].items[].id" type="integer">
      Order item ID
    </ResponseField>
    
    <ResponseField name="orders[].items[].order_id" type="integer">
      Associated order ID
    </ResponseField>
    
    <ResponseField name="orders[].items[].product_id" type="integer">
      Product ID
    </ResponseField>
    
    <ResponseField name="orders[].items[].product_name" type="string">
      Product name
    </ResponseField>
    
    <ResponseField name="orders[].items[].quantity" type="integer">
      Quantity ordered
    </ResponseField>
    
    <ResponseField name="orders[].items[].unit_price" type="number">
      Price per unit at time of order
    </ResponseField>
    
    <ResponseField name="orders[].items[].total_price" type="number">
      Total price for this item (unit_price × quantity)
    </ResponseField>
  </ResponseField>
</ResponseField>

<ResponseField name="count" type="integer">
  Total number of orders
</ResponseField>

### Example

```bash cURL
curl -X GET http://localhost:5000/api/orders \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

**Response (200 OK):**
```json
{
  "orders": [
    {
      "id": 1,
      "user_id": 1,
      "status": "paid",
      "subtotal": 1299.99,
      "tax": 104.0,
      "discount_amount": 0.0,
      "total": 1403.99,
      "discount_code": null,
      "created_at": "2024-01-15T10:00:00",
      "updated_at": "2024-01-15T10:05:00",
      "items": [
        {
          "id": 1,
          "order_id": 1,
          "product_id": 1,
          "product_name": "Laptop Pro 15",
          "quantity": 1,
          "unit_price": 1299.99,
          "total_price": 1299.99
        }
      ]
    }
  ],
  "count": 1
}
```

### Error Responses

**401 Unauthorized** - Missing or invalid token:
```json
{
  "msg": "Missing Authorization Header"
}
```

## GET /api/orders/\<id\>

Retrieve a specific order by ID.

### Request

**Method:** `GET`

**Path:** `/api/orders/<id>`

**Authentication:** JWT token required

**Headers:**
- `Authorization: Bearer <access_token>`

**Path Parameters:**

<ParamField path="id" type="integer" required>
  Order ID
</ParamField>

### Response

<ResponseField name="order" type="object">
  Order object with all details including items (same structure as orders array above)
</ResponseField>

### Example

```bash cURL
curl -X GET http://localhost:5000/api/orders/1 \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

**Response (200 OK):**
```json
{
  "order": {
    "id": 1,
    "user_id": 1,
    "status": "paid",
    "subtotal": 1299.99,
    "tax": 104.0,

    "items": [
      {
        "id": 1,

        "unit_price": 1299.99,
```

## POST /api/orders

Create a new order.

### Request

**Method:** `POST`

**Path:** `/api/orders`

**Authentication:** JWT token required

**Headers:**
- `Authorization: Bearer <access_token>`
- `Content-Type: application/json`

**Body Parameters:**

<ParamField path="items" type="array" required>
  Array of order items. Must include at least one item.
  
  <ParamField path="items[].product_id" type="integer" required>
    Product ID to order
  </ParamField>
  
  <ParamField path="items[].quantity" type="integer" default="1">
    Quantity to order
  </ParamField>
</ParamField>

<ParamField path="discount_code" type="string">
  Optional discount code to apply
</ParamField>

### Response

<ResponseField name="order" type="object">
  Created order object with all details including items
</ResponseField>

### Example

```bash cURL
curl -X POST http://localhost:5000/api/orders \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." \
  -H "Content-Type: application/json" \
  -d '{
    "items": [
      {
        "product_id": 1,
        "quantity": 1
      },
      {
        "product_id": 2,
        "quantity": 2
      }
    ],
    "discount_code": "SAVE10"
  }'
```

**Response (201 Created):**
```json
{
  "order": {
    "id": 2,
    "user_id": 1,
    "status": "pending",
    "subtotal": 1359.97,
    "tax": 108.8,
    "discount_amount": 135.99,
    "total": 1332.78,
    "discount_code": "SAVE10",
    "created_at": "2024-01-15T14:30:00",
    "updated_at": "2024-01-15T14:30:00",
    "items": [
      {
        "id": 2,
        "order_id": 2,
        "product_id": 1,
        "product_name": "Laptop Pro 15",
        "quantity": 1,
        "unit_price": 1299.99,
        "total_price": 1299.99
      },
      {
        "id": 3,
        "order_id": 2,
        "product_id": 2,
        "product_name": "Wireless Mouse",
        "quantity": 2,
        "unit_price": 29.99,
        "total_price": 59.98
      }
    ]
  }
}
```

### Error Responses

**400 Bad Request** - Missing items:
```json
{
  "error": "Order must include at least one item"
}
```

**400 Bad Request** - Insufficient stock:
```json
{
  "error": "Insufficient stock for Laptop Pro 15. Available: 5"
}
```

**401 Unauthorized** - Missing or invalid token:
```json
{
  "msg": "Missing Authorization Header"
}
```

**404 Not Found** - Product not found:
```json
{
  "error": "Product 999 not found"
}
```

## Order Processing

When an order is created:

1. **Validation**: The system validates that all products exist and have sufficient stock
2. **Calculation**: Subtotal, tax, and discounts are calculated automatically
3. **Stock Reduction**: Product stock is reduced by the ordered quantity
4. **Status**: Order is created with `pending` status
5. **Payment**: Use the Payments API to complete the order

The total is calculated as:
```
total = subtotal + tax - discount_amount
```
